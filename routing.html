Routing

Alexandros Frangos

Routing is the process of sending packets of data from origins to destinations, such as the Internet which involves selecting network paths to carry network traffic. Routing can be used with many networks including circuit switching and packet switching networks. This process is done by a device called a router.

Circuit Switching --> In circuit switching a tellecommunications network is implemented and all the data packets are send in order. As a result, when all the packets arrive at destination, a sorting of them isn't required.

 

Packet Switching --> In packet switching a message is divided to packets before it is sent. Each packet is transmitted individually and when all packets arrive at destination they are reassembled to form the original message.

Research at UCL

Did you know that starting from 1986 the Internet was faced with a series of congestion collapses? Especially in 1988 the Internet was saved by a congestion collapse by using an algorithm in which users’ machines could minimize their transmission rate when a network was detected to be overloaded.

Research at UCL involves thinking routing as a way in which computers in a computer network can control congestion. If users could be given a range of paths and they were induced to take the least congested path this would remove significant strains from the Internet’s current routing system.

Other Research

The Compact Routing problem (Center for Applied Internet Data Analysis):

Compact routing is the theory in which the fundamental limits for routing efficiency is studied and tries to make algorithms which meet those limits.

Input: A network G (a connected weighted graph)
Output: An algorithm that allows any source node to route messages to any destination node given the destination's network identifier.
Goal: Make routing algorithms that meet the fundamental limits for routing efficiency.
