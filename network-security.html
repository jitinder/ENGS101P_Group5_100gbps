Network Security

By Sidak Pasricha

petsIntroduction

In an interconnected world where practically devices are connected over a network, it is vital there are some measures taken to ensure security of sensitive data and prevent unauthorised access to any form of data. Network security is the process of taking these measures on the infrastructure that shapes the network to help create a secure and reliable network. If a network is compromised, there is a risk of misuse, malfunction, modification, destruction or leakage which can lead to major consequences.



Networks can be made secure through various methods, even combinations of methods. Listed below are some common methods to provide Network Security:
  • fingerprint Authentication
  • https Firewall
  • bug_report Anti-Virus
  • low_priority Encryption


Authentication refers to using one or more factors to ensure the person using the network is authorised to access data or be on the network.

There are three major types of Authentication:

  • One-Factor: There is only one detail required from the user in addition to their username. (e.g. a password)
  • Two-Factor: Another detail is used, usually something that the user has. (e.g. A Mobile-phone)
  • Three-Factor: The user is used to uniquely identify themselves using biometric data that is unique to them.


A firewall is an inbuilt protection software that enforces access policies such as what services are allowed to be accessed by the authenticated users.

While a firewall is effective in preventing unauthorised entities from using the network, it does not ensure protection from any malicious files or softwares.


As authentication or firewall are only useful when it comes to prevent unauthorised access to the network. There is a requirement for a software that protects the network against dangerous software like Viruses, Malware and Trojans that have the capability to compromise the network and data by corrupting or leaking files.

This is why Anti-Virus software is one of the most essential requirements in network security.


Mainly used in data transfer, Encryption is the method of "scrambling" data using algorithms so that the data cannot be interpreted if it is intercepted during the transfer process by an unauthorised entity. In order to decrypt or understand the data, a decryption key is required.

For example, consider the phrase "Hello World". It will be encrypted using one of the various algorithms and a decryption key will be created.
The encrypted phrase might look something like, "91hd4galsfcna0w187fg". Doesn't look remotely like "Hello World", does it.
When the person whom the message was intended for receives the encrypted phrase and decrypts it using the key, the phrase becomes "Hello World again" for the appropriate usage.

Fun Fact: This is used in Whatsapp. open a chat's settings and you will see that "End-to-End Encryption" is being used to ensure the chat remains between the intended people.


gradeResearch at UCL

Since one can not know how trustworthy the operating system or hardware of the e-commerce website we trust with our credit card information or other sensitive information is, the research is geared towards creating a secure system that neutralises the weaknesses during the creation of the network, basically, disregard unreliability of the parts used in creating the system.

The aim is to find clean design principles and to provide software toolkits, so that programmers can more easily reason about security for complex computer systems.

Securing RFID Chips in devices from close range data extraction [1]:
Researchers have developed a new type of radio frequency identification chip that is not easily hackable. If such chips were widely adopted, an identity thief couldn’t steal your credit card numberinformation by sitting next to you.

Creating new Algorithms for Cryptography (Encrypting and Decrypting data) [2]:
Researchers are also trying to fortify the standards of encryption currently in place to make it impossible to compromise security systems. This is done in order to keep upto date with modern advancements in Technology where brute forcing even the most secure techniques is done faster due to improved processing speeds and powerful systems like quantum computers.
