Internet of Things

By Zixuan Wang

pets Introduction

The internet of things is the network of physical items embedded with electronics, software, sensors, actuators and network connectivity which enable objectives to gather and exchange data. These devices collect useful data with the help of existing technologies and then autonomously flow the data between other devices [1].

These items represent a wide variety of devices such as heart monitoring implants, biochip transponders on farm animals, cameras streaming live feeds of wild animals in coastal waters, automobiles with built-in sensor or field operation devices that assist firefighters in search and rescue operations.

The research of internet of things is mainly focused on network architecture and applications of infrastructure management, medical and healthcare, media, transportation and metropolitan scale deployments. Below are some research conducted by MIT, basically are some applications：

1) CarTel: a mobile sensor network system developing vehicular network protocols, software, and services [2].

2) Cricket: an accurate indoor location system which is commercially available now [3].

3) Migrate: an end-to-end architecture for Internet host mobility, support for suspend/resume operations for mobile network applications to handle disconnections, and server failover [4].
